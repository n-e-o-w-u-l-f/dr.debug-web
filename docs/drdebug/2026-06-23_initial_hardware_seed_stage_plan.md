# Initial Hardware Seed Stage Plan

Date: 2026-06-23
Status: PROPOSAL_ONLY
Canonical: false
Review required: true

## Goal

Build a broad, evidence-based Dr.Debug seed for electronic devices, manufacturers, manuals, firmware, drivers, platforms, games, endpoints and relations.

## Safety prerequisite

Before any broad seed generation:

1. Verify repository context.
2. Verify target path.
3. Verify source hierarchy.
4. Avoid raw binary/manual downloads without source records.
5. Avoid canonical promotion without review.
6. Avoid generated-data shrink.
7. Avoid unbounded generation.
8. Avoid local-only build assumptions for GitHub Pages repos.
